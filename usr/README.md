# Silabus Hijaiyah Bersambung

Aplikasi edukatif interaktif berbasis Flutter yang menampilkan pokok dan sub pokok bahasan untuk materi **Membaca, Menulis, dan Membedakan Huruf Hijaiyah Bersambung**. Aplikasi ini didesain responsif untuk berbagai perangkat mulai dari ponsel genggam (mobile), tablet, hingga web/desktop dengan tampilan yang modern dan islami.

## Pokok Bahasan Utama

Aplikasi ini mencakup silabus sebagai berikut:
1. **Pengenalan Huruf Hijaiyah Bersambung**: Konsep menyambung huruf dan huruf yang tidak bisa disambung.
2. **Membaca Huruf Hijaiyah Bersambung**: Teknik membaca 2 hingga kata pendek yang tersambung.
3. **Menulis Huruf Hijaiyah Bersambung**: Teknik dasar menulis dari kanan ke kiri dan menebalkan huruf.
4. **Membedakan Huruf yang Berubah**: Analisis huruf yang memiliki perubahan drastis bentuk saat bersambung.
5. **Latihan dan Evaluasi**: Kuis membaca, dikte, dan evaluasi berkelanjutan.

## Alur Penggunaan Aplikasi (User Flow)
1. **Beranda (Home)**: Pengguna akan disajikan dengan tampilan *Header* yang informatif mengenai topik pembelajaran.
2. **Eksplorasi Pokok Bahasan**: Pengguna dapat melihat daftar pokok bahasan utama yang dikemas di dalam *Card*.
3. **Detail Sub Pokok**: Pengguna dapat menekan setiap pokok bahasan (ExpansionTile) untuk membuka dan membaca sub pokok materi yang relevan di bawahnya.

## Tumpukan Teknologi (Tech Stack)
* **Framework**: Flutter
* **Bahasa**: Dart
* **Desain UI**: Material Design 3

## Cara Menjalankan Aplikasi
1. Pastikan Flutter SDK telah terinstal di komputer Anda.
2. Salin *repository* proyek ini atau unduh *source code*.
3. Buka terminal pada folder proyek, lalu jalankan `flutter pub get` untuk memastikan semua paket termuat.
4. Jalankan aplikasi dengan menggunakan perintah:
   ```bash
   flutter run
   ```
